# 프로토타이핑 2 — 하네스 A/B (H2 품질 검증)

> "재미는 소재 유형과 각색 하네스 중 무엇이 좌우하는가"를 확인한다.

## 실험 설계 (2x2)
같은 문서를 엔진 v1(담백) / v2(썰)로 각각 돌려 4개 결과물을 만든다.

| | 기사 (GTX 등) | 판결문 (전세사기 등) |
|---|---|---|
| v1 담백 | A | C |
| v2 썰 | B | D |

## 진행
- [ ] 문서 2개 선정: 행정형 기사 1개 + 갈등형 판결문 1개
- [ ] 4개 조합 생성, 각 결과 저장 (제목 가리고 라벨만: A/B/C/D)
- [ ] 지인 3~5명에게 블라인드로 제시: "어느 글을 끝까지 읽고 싶나? 순위를 매겨달라"
- [ ] 추가 질문: "어느 글이 제일 '유튜브 썰' 같나?"

## 판정
- D > B > C > A 이면: 소재와 하네스 둘 다 효과 → 판결문 + v2로 확정
- B ≈ D 이면: 하네스가 지배적 → 소재 폭 넓게 유지 가능
- C ≈ D 이면: 소재가 지배적 → 하네스보다 소재 큐레이션에 투자
- 검증 리포트의 "불일치" 발생률도 엔진별로 기록 (썰 기법이 사실 왜곡을 늘리는지 감시)

## 판결문 구하는 곳
- 국가법령정보센터 판례(law.go.kr) / 대법원 종합법률정보 — 무료, 주요 판례
- 판결서 인터넷열람(scourt.go.kr) — 건당 1,000원, 확정 판결
- 주의: 비실명(A, B, 갑, 을) 표기 유지. 신원 추정 금지.
