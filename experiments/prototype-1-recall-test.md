# 프로토타이핑 1 — 회상 테스트 (H1 가치 검증)

> 코딩 0줄로 "이야기로 읽으면 정말 더 기억에 남는가"를 확인한다.
> 이게 통과 못 하면 서비스 존재 이유가 없으므로 가장 먼저 한다.

## 준비
- [ ] 사건 전개형 기사 3개 선정 (분쟁·정책·기업 사건 — 단순 시황 제외)
- [ ] 각 기사를 썰풀이(또는 손)로 이야기 버전 생성 + 사실 대조 확인
- [ ] 참가자 10명 → A그룹(원문 5명) / B그룹(이야기 5명) 무작위 배정

## 진행
1. 각자 배정된 버전을 읽게 한다 (시간 제한 없음, 완독 여부 기록)
2. 읽은 직후: "이해됐다" 1~5점 자가평가
3. **24시간 뒤**: 예고 없이 회상 퀴즈
   - "무슨 사건이었는지 설명해달라" (자유 회상)
   - 핵심 사실 3개 질문 (누가 / 무엇을 / 왜)

## 기록표
| 참가자 | 그룹 | 완독 | 직후 이해도 | 24h 자유회상(0~2) | 사실퀴즈(0~3) |
|---|---|---|---|---|---|
| P1 | A | | | | |

## 판정
- B그룹(이야기)의 24h 회상·퀴즈 점수가 체감될 만큼 우위 → H1 통과, H2로 진행
- 차이 없음 → 서사화 방식 재검토 (통찰 단락 강화? 유형 변경?)
- 주의: "재밌었다"는 측정하지 않는다. 기억에 남았는가만 본다.
